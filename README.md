# 🩺 Predictor de Diabetes — Árbol de Decisión

Modelo de Machine Learning para predecir si un paciente 
tiene diabetes según indicadores clínicos.

## Tecnologías
- Python
- Pandas
- Scikit-learn

## Dataset
UCI Pima Indians Diabetes Database — 768 registros  
Features: Glucosa, IMC, Edad, Historial familiar, Presión arterial

## Modelo
Árbol de Decisión (DecisionTreeClassifier) con evaluación 
mediante train_test_split (80% entrenamiento / 20% prueba)
